## 1. 로그인

![로그인 화면](manual-images/01-login.png)

- 접속 주소: `http://localhost:3000`
- **이메일 + 비밀번호**로 로그인합니다.
- 팀원 계정의 **초기 비밀번호는 `tms2026!`** 입니다.
- 하단의 팀원 이름(마승일 · 신선중 · 왕유림 · 맹미란 · 박재원)을 클릭하면 해당 계정으로 빠르게 전환됩니다.

## 3. 대시보드

![대시보드](manual-images/02-dashboard.png)

팀 전체의 **실시간 업무 현황**을 한눈에 보여주는 메인 화면입니다.

- **실시간 업무현황 (LIVE)** — 멤버별 현재 진행 업무와 진행률을 카드로 표시. 우측 상단에 현재 접속 인원 표시.
- **진행률 색상 코딩**: 0% ⚪ / ~25% 🔴 / ~50% 🟠 / ~75% 🟡 / ~99% 🔵 / 100% 🟢
- 우측 **[업무 부여]** 패널에서 새 업무를 바로 생성합니다.
  - **업무 대분류 선택**: `롱` 또는 `쇼츠` — 대시보드에서는 이 두 종류만 직접 생성합니다.
  - **`프로젝트`는 대시보드에서 생성 불가** → [프로젝트] 탭에서 먼저 만들어야 합니다.
  - **[상세 입력]** 으로 담당자 · 제목 · 마감일 등을 지정합니다.
- 상단 우측 **[회의 시작]** 버튼, 태스크·프로젝트 검색창 제공.

## 4. 프로젝트

![프로젝트 목록](manual-images/03-projects.png)

진행 중인 프로젝트를 카드 형태로 관리합니다.

- 각 카드: 프로젝트명 · 상태(진행중) · 기간/마감 D-day · 진행률 바 · 참여 멤버.
- **[+ 새 프로젝트]** 카드로 프로젝트를 생성합니다. (생성 마법사 → 담당자/참여자 설정)
- 프로젝트 단위 업무는 반드시 **여기서 먼저 생성·설정**한 뒤 담당자/참여자를 추가합니다.

## 5. 프로젝트 상세 · 칸반 보드

![프로젝트 상세 상단](manual-images/04-project-detail-top.png)

프로젝트 카드를 클릭하면 상세 화면으로 이동합니다.

- **상단**: 프로젝트명 · 상태 · 설명 · 기간 · 참여자 · 진행률(태스크 기준).
- **[멤버 관리]** / 상태(활성) 토글 / 우측 월간 미니 캘린더.
- **[업무설명 보기]** · **[기획서]** 링크.
- **🤖 AI 소통 요약** — 대화가 쌓이면 [지금 요약] 으로 핵심을 자동 정리.
- **커뮤니케이션** 패널 — `@멘션`으로 멤버를 호출해 메시지를 주고받습니다.

![프로젝트 칸반 보드](manual-images/05-project-kanban.png)

- **태스크 보드 (칸반)**: `할일 → 진행중 → 검토중 → 완료` 4단계.
- **카드를 드래그**하면 상태가 변경되고, **클릭**하면 상세 보기/수정.
- **[+ 태스크 추가]** 로 새 태스크 생성 (대분류 category / 산출물 요구 taskType 포함).
- 카드의 색 배지(긴급/보통/낮음)는 우선순위, 우측 숫자는 마감/진행 정보.

## 6. 캘린더

![캘린더](manual-images/06-calendar.png)

월별로 업무 진행 현황과 프로젝트 타임라인을 봅니다.

- 상단 탭: **업무 / 근무 / 회의** 로 보기 전환.
- ◀ ▶ 로 월 이동, 우측 검색으로 업무·담당자 필터.
- 일자 셀에 업무 일정이 색 막대로 표시됩니다.
- 하단 **프로젝트 타임라인** 에서 프로젝트별 기간을 한눈에 확인.

## 7. 내 활동

![내 활동](manual-images/07-activity.png)

개인 업무 중심 화면입니다.

- **⏳ 마감 임박 (D-1)** 배너로 임박 업무 강조.
- **나의 업무** — 내가 맡은 업무 목록, 각 항목 **[오늘 하기]** 로 오늘의 업무에 추가.
- **요청한 업무 / 요청받은 업무** — 내가 보낸/받은 업무와 수락 상태·진행률.
- **✅ 오늘의 업무 (체크리스트)** — **핵심 기능**
  - 체크 = 대시보드에 즉시 **진행중** 표시 + **시작 시각 기록**.
  - **[종료]** = 종료 시각 기록 + 산출물 입력 폼.
- **🗓 오늘 스케줄 (계획 ‖ 실제)** — 업무를 타임라인에 드래그해 계획/실제 시간을 기록.
- **멘션 & 소통 피드** — 나를 호출한 멘션과 알림 모음.

## 8. 요청 업무 관리

![요청 업무 관리](manual-images/08-requests.png)

내가 요청한 업무의 처리 현황을 표로 분석합니다.

- 컬럼: 업무 · 담당자 · 요청일 · 완료일 · 소요 · 마감대비 · 재작업 · 등급 · AI 한줄평.
- 상단 **담당자 / 월(완료 기준)** 필터로 좁혀 볼 수 있습니다.
- 우측 상단 총 건수 표시.

## 9. 메시지 (DM)

![메시지 DM](manual-images/09-dm.png)

사내 메신저 영역입니다.

- **전체 공지** · **1:1** · **그룹** 대화를 지원합니다.
- 좌측: 대화 목록 + 검색 + **[+ 그룹]** 생성, 핀(📌) 고정.
- **새 1:1 대화** 영역에서 멤버를 골라 바로 대화를 시작합니다.

## 10. 설정

![설정](manual-images/10-settings.png)

내 계정·휴가 관리 화면입니다. (관리자 계정은 **관리 모드** 추가 표시)

- **멤버 관리** (관리자) — 멤버별 직책 · 담당업무 · 연차잔여 수정 후 [저장].
- **내 휴가** — 종류(연차/반차/반반차) · 사유 · 시작/종료일 입력 후 **[휴가 신청]**.
  - 관리자 승인 시 종류별로 잔여 연차에서 자동 차감(연차 −1 / 반차 −0.5 / 반반차 −0.25).
- **전 멤버 휴가 (승인/반려)** (관리자) — 신청된 휴가를 **[승인] / [반려]** 처리.

## 11. 핵심 용어 정리

| 용어 | 의미 |
|------|------|
| **대분류 (category)** | 업무 종류 — `롱(long)` / `쇼츠(shorts)` / `프로젝트(project)` |
| **산출물 요구 (taskType)** | `report` / `video` / `both` / `none` — 대분류와 별개 축 |
| **담당자 (owner)** | 프로젝트의 역할별 책임자 |
| **참여자 (participant)** | 프로젝트의 협업 멤버 |
| **부여자 (assigner)** | 업무를 부여한 사람 |
| **담당자 (assignee)** | 업무를 수행하는 사람 |

> ※ 롱·쇼츠는 대시보드에서 직접 생성, **프로젝트는 [프로젝트] 탭에서만 생성** 가능합니다.
