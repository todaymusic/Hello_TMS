TMS 사용 매뉴얼

사내 업무 관리 플랫폼(TMS)의 모든 화면과 기능을 단계별로 안내합니다. 콘텐츠 제작팀의 롱·쇼츠·프로젝트 워크플로우에 맞춘 슬랙형 협업 도구입니다. 아래 화면 캡쳐는 실제 TMS 화면입니다.

접속 · http://localhost:3000초기 비밀번호 · tms2026!대상 · 전 팀원
1

로그인

이메일과 비밀번호로 TMS에 접속합니다.

  • 접속 주소 http://localhost:3000 으로 들어가면 자동으로 로그인 화면이 나옵니다.
  • 회사 발급 이메일 + 비밀번호를 입력합니다. 팀원 계정의 초기 비밀번호는 tms2026! 입니다. (최초 로그인 후 설정에서 변경 권장)
  • 하단 팀원 이름(마승일·신선중·왕유림·맹미란·박재원)을 클릭하면 해당 계정 이메일이 자동 입력되어 빠르게 전환됩니다.
✅
처음 쓰는 분께 — 로그인 후 왼쪽 세로 막대(사이드바)가 모든 메뉴의 출발점입니다. 다음 2번 항목을 먼저 읽어보세요.
2

화면 구성 (사이드바)

왼쪽 세로 사이드바의 아이콘으로 6개 주요 영역을 오갑니다. 어느 화면에서든 항상 보입니다.

아이콘메뉴이런 걸 합니다
▦ 그리드대시보드팀 전체 실시간 업무 현황 · 새 업무(롱/쇼츠) 부여
📅 달력캘린더월별 업무·근무·회의 일정과 프로젝트 타임라인
▥ 보드프로젝트프로젝트 생성·멤버 관리·칸반 보드
👤 사람내 활동내 업무·오늘 할 일 체크리스트·멘션·근태 스케줄
💬 말풍선메시지(DM)1:1·그룹 대화·전체 공지
⚙ 톱니설정내 계정·휴가 신청 / (관리자) 멤버·휴가 승인
🔔
우측 하단 종 아이콘은 알림 센터입니다. 숫자 배지는 읽지 않은 멘션·업무 알림 개수입니다.
3

대시보드

팀 전체가 "지금 무엇을 하고 있는지" 실시간으로 보여주는 메인 화면입니다.

실시간 업무현황 (LIVE)

  • 멤버별 카드에 현재 진행 중인 업무와 진행률이 표시됩니다. 누가 무엇을 하는지 새로고침 없이 실시간 갱신됩니다.
  • 우측 상단 접속 N명으로 현재 온라인 인원을 알 수 있습니다.
  • 진행률은 경과에 따라 색이 바뀝니다: 0% 회색 ~25% 빨강 ~50% 주황 ~75% 노랑 ~99% 파랑 100% 초록

업무 부여 (롱 · 쇼츠)

  • 우측 패널에서 대분류(롱/쇼츠)를 고르고 [상세 입력]으로 담당자·제목·마감일·산출물 요구를 지정해 업무를 만듭니다.
  • 부여된 업무는 담당자의 내 활동에 '요청받은 업무'로 도착합니다.
⚠️
프로젝트는 여기서 못 만듭니다. 롱·쇼츠만 대시보드에서 직접 생성합니다. 프로젝트 단위 업무는 [프로젝트] 탭에서 먼저 만들어야 합니다.
4

프로젝트

진행 중인 프로젝트를 카드로 모아 관리합니다. 프로젝트는 반드시 이 화면에서 생성합니다.

  • 각 카드: 프로젝트명 · 상태 · 기간/마감 D-day · 진행률 바 · 참여 멤버 아바타를 한눈에 보여줍니다.
  • [＋ 새 프로젝트] 카드를 누르면 생성 마법사가 열립니다 → 이름·기간 입력 후 담당자(owner)와 참여자(participant)를 지정합니다.
  • 카드를 클릭하면 프로젝트 상세(5번)로 이동합니다.
💡
담당자 vs 참여자 — 담당자는 역할별 책임자(예: 영상 담당, 디자인 담당), 참여자는 함께 협업하는 멤버입니다. 두 역할은 혼용하지 않습니다.
5

프로젝트 상세 · 칸반 보드

프로젝트 카드를 클릭하면 진행률·소통·태스크를 한 화면에서 관리합니다.

상단 정보

  • 진행률(태스크 기준) · 기간 · 참여자 · 상태 토글 · 우측 월간 미니 캘린더.
  • [업무설명 보기]와 [기획서] 링크로 관련 문서를 엽니다.
  • 🤖 AI 소통 요약 — 커뮤니케이션 대화가 길어지면 [지금 요약] 버튼으로 핵심을 자동 정리합니다.
  • 커뮤니케이션 패널에서 @멘션으로 멤버를 호출해 프로젝트 안에서 바로 대화합니다.

태스크 보드 (칸반)

  • 업무는 할일 → 진행중 → 검토중 → 완료 4단계로 흐릅니다.
  • 카드를 드래그하면 상태가 바뀌고, 카드를 클릭하면 상세 보기/수정 창이 열립니다.
  • [＋ 태스크 추가]로 새 태스크를 만들 때 대분류(category)와 산출물 요구(taskType)를 함께 지정합니다.
  • 카드 색 배지(긴급 높음 보통)는 우선순위입니다.
6

캘린더

월별로 업무·근무·회의 일정과 프로젝트 타임라인을 봅니다.

  • 상단 업무 / 근무 / 회의 탭으로 보기를 전환합니다.
  • ◀ ▶ 로 월을 이동하고, 우측 검색으로 업무·담당자를 필터링합니다.
  • 각 날짜 칸에 일정이 색 막대로 표시됩니다(담당자·제목 포함).
  • 하단 프로젝트 타임라인에서 프로젝트별 기간을 가로 막대로 한눈에 봅니다.
7

내 활동

'나' 중심의 화면입니다. 오늘 할 일을 시작/종료하고, 받은 멘션과 근태를 관리합니다.

  • ⏳ 마감 임박 배너로 임박한 업무를 가장 위에서 강조합니다.
  • 나의 업무 — 내가 맡은 업무. 각 항목의 [오늘 하기]로 '오늘의 업무'에 올립니다.
  • 요청한 업무 / 요청받은 업무 — 내가 보낸/받은 업무와 수락 상태·진행률.
  • ✅ 오늘의 업무 (체크리스트) — TMS의 핵심 기능입니다:
    • 체크 = 시작 → 대시보드에 즉시 '진행중' 표시 + 시작 시각 기록.
    • [종료] → 종료 시각 기록 + 산출물 입력 폼이 뜹니다.
  • 🗓 오늘 스케줄 (계획 ‖ 실제) — 업무를 타임라인에 드래그해 계획/실제 근무 시간을 기록합니다.
  • 멘션 & 소통 피드 — 나를 호출한 멘션·알림을 모아 보고 [모두 읽음] 처리합니다.
8

요청 업무 관리

내가 요청한 업무들의 처리 현황(소요·마감·재작업·등급)을 표로 분석합니다.

  • 컬럼: 업무 · 담당자 · 요청일 · 완료일 · 소요 · 마감대비 · 재작업 · 등급 · AI 한줄평.
  • 상단 담당자 / 월(완료 기준) 필터로 원하는 범위만 봅니다.
  • 업무가 완료되면 소요시간·마감 대비·재작업 횟수가 채워지고, AI 한줄평으로 품질 피드백이 붙습니다.
  • 우측 상단에 총 건수가 표시됩니다.
9

메시지 (DM)

사내 메신저 영역입니다. 1:1·그룹·전체 공지를 한 곳에서 처리합니다.

  • 전체 공지 · 1:1 · 그룹 대화를 지원합니다.
  • 좌측: 대화 목록 + 검색 + [＋ 그룹] 생성, 핀(📌)으로 중요한 대화를 위로 고정합니다.
  • 새 1:1 대화 영역에서 멤버를 골라 바로 대화를 시작합니다.
10

설정

내 계정·휴가를 관리합니다. 관리자 계정에는 '관리 모드' 기능이 추가로 보입니다.

모든 멤버 공통

  • 내 휴가 — 종류(연차/반차/반반차)·사유·시작/종료일을 입력하고 [휴가 신청]을 누릅니다.
  • 관리자 승인 시 종류별로 잔여 연차에서 자동 차감됩니다: 연차 −1 / 반차 −0.5 / 반반차 −0.25.

관리자 전용 (관리 모드)

  • 멤버 관리 — 멤버별 직책·담당업무·연차잔여를 수정하고 [저장].
  • 전 멤버 휴가 — 신청된 휴가를 [승인] / [반려] 처리합니다.
＊

핵심 용어 정리

TMS 전체에서 일관되게 쓰이는 용어입니다.

용어의미
대분류 (category)업무 종류 — 롱(long) / 쇼츠(shorts) / 프로젝트(project)
산출물 요구 (taskType)report / video / both / none — 대분류와 별개의 축으로 조합합니다
담당자 (owner)프로젝트의 역할별 책임자
참여자 (participant)프로젝트의 협업 멤버
부여자 (assigner)업무를 부여한 사람
담당자 (assignee)업무를 수행하는 사람
📌
업무 생성 규칙 — 롱·쇼츠는 대시보드에서 직접 생성, 프로젝트는 [프로젝트] 탭에서만 생성할 수 있습니다.
